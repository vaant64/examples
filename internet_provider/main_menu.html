Основное меню


1) Показатть сведения о заказах оборудования за указанный год по форме:
уникальный шифр, название, общее количество в заказах, общая стоимость.
2) Показать все сведения о работе монтажников за указанный год по форме:
табельный номер, фамилия, общее количество накладных, общая стоимость работ по накладным.
3) Показать сведения о монтажнике, который работает в компании дольше всех остальных.
4) Показать сведения о монтажнике, оформившем самую дорогую накладную в 2017 году.
5) Показать сведения о монтажниках, не составивших ещё ни одной накладной.
6) Показать сведения о монтажниках, не составивших накладных в марте 2017 года.
Составить отчёт содержащий информацию о выполненных работах за указанный год и месяц.
Вывести отчёт о выполненных работах за заданный период.
